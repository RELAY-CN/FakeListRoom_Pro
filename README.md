# FakeListRoom_Pro
用于在 Rusted Warfare 的列表上通过 * 方式实现虚假的房间 (更简单/更多)  
游戏版本: 1.15 (176)  

默认方案
> 创建一直在的房间 (保活)  

**不为此负责**  

## Tags
`红队` `攻击性`   

## 使用
### 单个房间
```text
bash fake.sh add ip:port@type
```
例子
```text
bash fake.sh add 1.1.1.1:1111@socks5
```
### 多个房间
```text
bash fakes.sh
```
> 需要提前准备端口代理 proxy.txt

## 其他
我们**不认为**长期置顶列表是**好事**, 我们应该释放出来给更有需要的  
建议遵循自带逻辑, 24*7 服务器不应该, 也不应当长期置顶恶心正常玩家
